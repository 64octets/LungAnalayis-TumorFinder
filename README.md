Program designed to look at X-ray images of Lungs, to analyse and identify tumors. Developed in Matlab, uses custom filter and threshold finding. 

# Tutorial 
1. Download Matlab
2. Run Function Command "lungAnalysis('***')"  ** = Image Location/Name
